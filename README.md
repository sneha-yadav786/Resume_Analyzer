📄 Resume Analyzer Web App
🚀 Overview
This project is a Resume Analyzer Web Application that helps users evaluate their resumes based on a selected job role. It provides an ATS-like score along with detailed feedback to improve resume quality.
The goal of this project is to simulate how real-world Applicant Tracking Systems (ATS) work and help users optimize their resumes for better job opportunities.

✨ Features


📂 Upload resume (PDF format)


🎯 Select target job role


📊 ATS-style scoring system


✅ Matching skills detection


❌ Missing skills identification


💡 Personalized feedback & suggestions


🔍 Keyword analysis using Regular Expressions


📈 Score improvement based on impactful metrics (like numbers, achievements)
